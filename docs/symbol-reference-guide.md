# 🔣 符号参考指南 - CursorRIPER♦○ Lite

*v1.0.0 | 创建: 2025-05-15*

本文档提供了CursorRIPER♦○ Lite框架中使用的所有符号表示法的参考。

## 🔍 几何符号

| 符号 | 名称 | 描述 |
|------|------|------|
| ◊ | 钻石形 | RIPER模式 (◊₁-◊₅) |
| ▲ | 三角形 | 项目阶段 (▲₁-▲₄) |
| ○ | 圆形 | 内存系统 |
| ◆ | 实心钻石 | 安全协议 |
| □ | 方形 | 内存文件 (□₁-□₅) |
| χ | chi | 交叉引用 |

## 📊 下标

下标用于表示特定实例或变体：
- ◊₁, ◊₂, ◊₃, ◊₄, ◊₅ - 五个RIPER模式
- ▲₁, ▲₂, ▲₃, ▲₄ - 四个项目阶段
- □₁, □₂, □₃, □₄, □₅ - 五个内存文件
- ◆₁, ◆₂, ◆₃, ◆₄ - 安全协议
