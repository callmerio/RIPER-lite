# 主任务清单

## 📋 任务管理说明

本文档记录 RIPER-lite 项目的主要任务和执行状态。任务按照 RIPER 方法论进行分类和管理。

### 任务状态说明

- `[ ]` **planning**: 计划中，尚未开始
- `[/]` **in_progress**: 进行中，正在执行
- `[x]` **completed**: 已完成，通过验收
- `[-]` **blocked**: 阻塞中，等待解决
- `[!]` **cancelled**: 已取消，不再执行

### 优先级说明

- 🔴 **High**: 高优先级，紧急重要
- 🟡 **Medium**: 中优先级，重要但不紧急
- 🟢 **Low**: 低优先级，可延后处理
